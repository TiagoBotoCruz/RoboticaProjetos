# RoboticaProjetos
Programas e games para jovens e adultos, para o básico de robótica
É um programa web feito em html , css  e javascript.
em que o usuário vai fazer os desenhos segundo é pedido pelo próprio programa.
O usuário faz o desenho pintando os quadrados da tabela.
Ele pode mudar a cor do pixel.
Usando input do tipo color.
